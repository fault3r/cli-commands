# mv

**Move** > move or rename files & directories.
---

` mv [OPTION]... [SOURCE] [DESTINATION] `
---

| **OPTION** | description |
|:---:|:---:|
| -v, --verbose | verbose mode |
| -i, --interactive | interactive mode	 |
| -f, --force | force mode |
| -n, --no-clobber | no overwrite mode |
| -u | update mode |

## Examples:
` mv -vuf test/ /home/ `
