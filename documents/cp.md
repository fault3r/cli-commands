# cp

**Copy** > copy files & directories.
---

` cp [OPTION]... [SOURCE] [DESTINATION] `
---

| **OPTION** | description |
|:---:|:---:|
| -r, --recursive | recursively mode |
| -v, --verbose | verbose mode |
| -i, --interactive | interactive mode	 |
| -f, --force | force mode |
| -n, --no-clobber | no overwrite mode |
| -u | update mode |

## Examples:
` cp -rv test/ /home/ `
