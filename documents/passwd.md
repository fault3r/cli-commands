# passwd

**Password** > change or remove user account password.
---

` passwd [OPTION] [USERNAME] `
---

| **OPTION** | description |
|:---:|:---:|
| -d, --delete | delete the password of user account |

### don't pass ` [USERNAME] ` to change current user account password.

## Examples:
` passwd `

` sudo passwd t-user `

` sudo passwd -d t-user `
