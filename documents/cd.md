# cd

**Change Diredctory** > change the shell working directory.
---

`cd [PATH]`
---

| **PATH** | description |
|:---:|:---:|
| / | root |
| ~ | home |
| ./ | current path |
| ../ | parent path |
| .. | back to parent |
| .. | back to parent |
| \ | use as space |
