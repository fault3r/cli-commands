# cd

**Change Diredctory** > change the shell working directory.
---

`cd [PATH]`
---

| **PATH** | description |
|:---:|:---:|
| | home directory |
| ~ | home directory |
| / | root directory |
| ./ | current path |
| ../ | parent path |

### use ` \ ` as space, ` cd /home/linux\ terminal/ `

## Examples
` cd ../../home/ `

` cd /usr/share/ `
