# ls

list information about the files (the current directory by default).
---

` ls [OPTION]... [FILE]... `
---

| **OPTION** | description |
|:---:|:---:|
| -l | view in list view mode |
| -a | display all hidden files and directories |

---

### create this alias ` la='ls -a' ` <br>
#### use these commands `ls`, `ls -l`, `la`, `la -l`

## Examples
` ls -l `

` ls -la / /home `

` ls -l file.txt `
