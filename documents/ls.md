# ls

list information about the files (the current directory by default).
---

` ls [OPTION]... [FILE]... `
---

| **OPTION** | description |
|:---:|:---:|
| -l | list mode |
| -a | all hidden files and directories included |

---

### create this alias ` la='ls -a' ` <br>
#### use these `ls`, `ls -l`, `la`, `la -l`

## Examples:
` ls -l `

` ls -la / /home `

` ls -l file.txt `
