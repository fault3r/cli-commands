# rm

**Remove** > remove files & directories.
---

` rm [OPTION]... [FILE]... `
---

| **OPTION** | description |
|:---:|:---:|
| -r, --recursive | recursively mode |
| -v, --verbose | verbose mode |
| -i, --interactive | interactive mode	 |
| -f, --force | force mode |

## Examples:
` rm -riv test/ `
